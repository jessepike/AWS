# Intelligence Layer — Intent

## What

A unified intelligence substrate for the Agentic Work System that makes every session smarter than the last. Two complementary stores — Knowledge Base (what we've learned) and Memory (what we've decided and what's true) — fed by automated capture pipelines, governed as critical infrastructure, and accessible from any AI client.

## Why

Without persistent, accessible intelligence, every agent session starts partially blind. Learnings re-discovered, decisions re-made, patterns re-identified. The system doesn't compound — it resets. The intelligence layer is what turns a collection of independent tools into a learning system.

## Success Looks Like

Agents start sessions with accumulated context. Learnings captured in one session are available in the next. Decisions made last week are surfaced this week. Content discovered by automated pipelines connects to operational context. The system gets measurably smarter over time.

## Constraints

- Solo developer, personal scale — not enterprise
- macbook2014 hosting for 3-6 months (temporary, architecture must be portable)
- Zero marginal API cost (subscriptions only)
- Must work with Claude Code, Codex, and ChatGPT (multi-client)
- Must fit within AWS governance (standards, specs, Forge ownership)
